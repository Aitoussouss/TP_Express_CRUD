# 📦 TP Express CRUD avec EJS

## 📖 Description

Ce projet est une application CRUD simple développée avec **Node.js**, **Express**, **MongoDB** et **EJS**.
Elle permet de gérer une liste de produits (ajout, affichage, modification, suppression).

---

## 🚀 Technologies utilisées

* Node.js
* Express.js
* MongoDB (Mongoose)
* EJS (Template Engine)
* Bootstrap (Design)

---

## ⚙️ Installation

1. Cloner le projet :

```bash
git clone https://github.com/Aitoussouss/TP_Express_CRUD.git
```

2. Accéder au dossier :

```bash
cd tp-crud-ejs
```

3. Installer les dépendances :

```bash
npm install
```

---

## ▶️ Lancer le projet

```bash
node server.js
```

Puis ouvrir dans le navigateur :

```
http://localhost:3000
```
